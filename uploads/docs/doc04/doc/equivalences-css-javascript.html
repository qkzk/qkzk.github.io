Tables des équivalences des propriétés CSS/Javascript

Rappel : pour modifier le style d’un élément ayant comme id monElement et donner la valeur laValeur à sa propriété laPropriete il faut utiliser la syntaxe suivante : 

document.getElementById(monElement).style.laPropriete = "laValeur";
