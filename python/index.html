RustPython

RustPython est un interpréteur Python écrit en Rust. Cette démonstration est compilée depuis Rust vers WebAssembly afin de fonctionner dans le navigateur.
Ce projet est récent et nécessite un navigateur moderne. À ce jour, sur mobile il fonctionne sous Firefox mais pas sous Chrome.
